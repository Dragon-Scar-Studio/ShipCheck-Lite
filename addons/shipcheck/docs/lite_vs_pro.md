# Lite vs Pro

ShipCheck Lite is free and should feel useful, not crippled. It catches common release-breaking mistakes inside the editor.

ShipCheck Pro is for teams and commercial releases. It adds automation, reports, baselines, CI modes, advanced scanners, and stricter quality workflows.

| Feature | Lite | Pro |
| --- | --- | --- |
| Editor dock | Yes | Yes |
| Missing scripts/resources | Yes | Yes |
| Case-sensitive path checks | Yes | Yes |
| Project settings checks | Yes | Yes |
| InputMap undefined actions | Yes | Yes |
| Debug leftovers | Yes | Yes |
| Basic release risk checks | Yes | Yes |
| Markdown report | Yes | Yes |
| Scene connection scanner | No | Yes |
| Performance hot-path scanner | No | Yes |
| Stale UID scanner | No | Yes |
| Advanced GDScript style/quality | No | Yes |
| JSON/HTML reports | No | Yes |
| Baselines/new issue tracking | No | Yes |
| CI fail modes | No | Yes |
